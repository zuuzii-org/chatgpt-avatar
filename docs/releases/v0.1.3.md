# ChatGPT Skin Studio 0.1.3 Public Beta

This patch makes the menu-bar control feel native to macOS: smaller, calmer, and focused on the actions people actually need.

## Download

- `ChatGPT-Skin-Studio-0.1.3.dmg` — signed and notarized macOS installer.
- `ChatGPT-Skin-Studio-0.1.3.zip` — signed Sparkle update archive.
- `appcast.xml` — EdDSA-signed Sparkle update feed.
- `SHA256SUMS.txt` — SHA-256 checksums for every published artifact.

## What’s new

- Replaced the custom menu-bar window with a compact native macOS menu: system row density, keyboard behavior, hover feedback, and automatic light/dark appearance.
- Reduced the menu to the current or selected theme, theme-library entry point, apply/switch/restore action, update check, About, and Quit.
- Removed build timestamps and verbose in-menu status text from the primary control surface; version information remains available in About.
- Replaced the diagonal Skin Peel mark with a monochrome stacked-card Template Image that better reads as a theme surface and avoids a prohibited-symbol appearance.

## Important

- ChatGPT Skin Studio is unofficial and is not affiliated with or endorsed by OpenAI.
- The Controller UI is currently available in Simplified Chinese.
- The first apply and native restore require explicit confirmation to restart ChatGPT. Save active work first.
- Compatibility with future ChatGPT releases is evaluated at runtime and is not guaranteed.
- This beta was verified with automated unit, contract, build, signature, notarization, package, checksum, and update-feed checks. Production E2E that would interrupt the user’s active ChatGPT session was intentionally not run for this release.
- Several bundled fan themes reference third-party franchises. A disclaimer does not replace permission from the relevant rights holders; see `THIRD_PARTY_NOTICES.md`.

# ChatGPT Skin Studio 0.1.3 Public Beta（中文）

这是一次菜单栏体验优化：更紧凑、更克制，并把注意力放回用户真正需要的操作。

## 下载文件

- `ChatGPT-Skin-Studio-0.1.3.dmg`：已签名并完成 notarization 的 macOS 安装包。
- `ChatGPT-Skin-Studio-0.1.3.zip`：经过签名的 Sparkle 更新包。
- `appcast.xml`：使用 EdDSA 签名的 Sparkle 更新源。
- `SHA256SUMS.txt`：所有发布产物的 SHA-256 校验值。

## 本次更新

- 自绘菜单栏窗口改为紧凑的原生 macOS 菜单：使用系统行高、键盘行为、悬停反馈以及浅色/深色外观。
- 菜单收敛为当前/所选主题、主题库入口、应用/切换/恢复、检查更新、关于和退出。
- 主操作面板不再展示构建时间和冗长状态；版本信息仍可在“关于”中查看。
- 对角 Skin Peel 图标改为单色双层卡片 Template Image，更贴近“主题表面”的语义，也避免被误读为禁止符号。

## 重要说明

- ChatGPT Skin Studio 是非官方项目，与 OpenAI 没有隶属或认可关系。
- Controller 界面当前仅提供简体中文。
- 首次应用皮肤和恢复原生界面需要明确确认并重启 ChatGPT，请先保存正在进行的工作。
- 对未来 ChatGPT 版本的兼容性由运行时探测决定，不作永久兼容承诺。
- 本测试版已完成自动化单元、契约、构建、签名、公证、打包、checksum 与更新源验证。由于生产 E2E 会中断用户当前的 ChatGPT 会话，本次按用户要求未运行该项测试。
