# ChatGPT Skin Studio 0.1.0 Public Beta

The first public beta of ChatGPT Skin Studio brings reversible, full-surface themes to the verified ChatGPT desktop app on macOS.

## Download

- `ChatGPT-Skin-Studio-0.1.0.dmg` — signed and notarized macOS installer.
- `ChatGPT-Skin-Studio-0.1.0.zip` — signed Sparkle update archive.
- `appcast.xml` — EdDSA-signed Sparkle update feed.
- `SHA256SUMS.txt` — SHA-256 checksums for every published artifact.

## Highlights

- New Chroma Peel App Icon across Finder and Launchpad sizes.
- Home Full, Thread Core, and conservative token-only modes.
- Ten bundled release themes plus local image-theme import.
- Runtime structural compatibility checks instead of a fixed ChatGPT version allowlist.
- No-restart switching while an already managed ChatGPT session remains active.
- Safe cleanup and native-interface recovery on stable incompatibility or runtime failure.
- Sparkle 2.9.2 update checks with a dedicated EdDSA signing key.

## Important

- ChatGPT Skin Studio is unofficial and is not affiliated with or endorsed by OpenAI.
- The Controller UI is currently available in Simplified Chinese.
- The first apply and native restore require explicit confirmation to restart ChatGPT. Save active work first.
- Compatibility with future ChatGPT releases is evaluated at runtime and is not guaranteed.
- This beta was verified with automated unit, contract, build, signature, notarization, package, checksum, and update-feed checks. Production E2E that would interrupt the user's active ChatGPT session was intentionally not run for this release.
- Several bundled fan themes reference third-party franchises. A disclaimer does not replace permission from the relevant rights holders; see `THIRD_PARTY_NOTICES.md`.

# ChatGPT Skin Studio 0.1.0 Public Beta（中文）

这是 ChatGPT Skin Studio 的首个公开测试版，可在 macOS 上为经过验证的 ChatGPT 桌面 App 应用可回退的完整界面皮肤。

## 下载文件

- `ChatGPT-Skin-Studio-0.1.0.dmg`：已签名并完成 notarization 的 macOS 安装包。
- `ChatGPT-Skin-Studio-0.1.0.zip`：经过签名的 Sparkle 更新包。
- `appcast.xml`：使用 EdDSA 签名的 Sparkle 更新源。
- `SHA256SUMS.txt`：所有发布产物的 SHA-256 校验值。

## 主要内容

- 新增 Chroma Peel 桌面 App 图标，覆盖 Finder 与 Launchpad 所需尺寸。
- 支持 Home Full、Thread Core 和克制的 token-only 模式。
- 首发包内置 10 套主题，并支持导入本地图片主题。
- 通过运行时结构探测判断兼容性，不绑定固定 ChatGPT 版本白名单。
- 受管 ChatGPT 会话激活后，可无重启切换主题。
- 遇到稳定不兼容或运行失败时执行安全清理并尝试恢复原生界面。
- 集成 Sparkle 2.9.2，并使用项目专用 EdDSA key 验证更新。

## 重要说明

- ChatGPT Skin Studio 是非官方项目，与 OpenAI 没有隶属或认可关系。
- Controller 界面当前仅提供简体中文。
- 首次应用皮肤和恢复原生界面需要明确确认并重启 ChatGPT，请先保存正在进行的工作。
- 对未来 ChatGPT 版本的兼容性由运行时探测决定，不作永久兼容承诺。
- 本测试版已完成自动化单元、契约、构建、签名、公证、打包、checksum 与更新源验证。由于生产 E2E 会中断用户当前的 ChatGPT 会话，本次按用户要求未运行该项测试。
- 部分内置 fan theme 涉及第三方作品；免责声明不能替代权利人许可，详见 `THIRD_PARTY_NOTICES.md`。
